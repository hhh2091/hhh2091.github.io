About

我目前在中国科学院大学攻读计算机应用技术博士学位。我的研究以强化学习为主线，关注在复杂约束环境中构建可训练、可泛化、可解释的决策模型，尤其面向长时序路径规划、连续控制和真实执行反馈驱动的策略优化问题。

近阶段的工作主要沿两条线展开。一条是面向平流层飞艇长期驻留任务的强化学习路径规划，研究如何将未来环境预报、时空表示和策略优化统一到同一框架中；另一条是面向 OpenROAD 等专业工具链的领域大模型智能体，探索利用真实执行结果、奖励设计与后训练方法提升代码智能体的可执行性与自修复能力。

Research

Reinforcement Learning for Stratospheric Airship Path Planning

Ongoing

面向平流层飞艇长期驻留任务，我构建了基于 PPO 的 STF-RL 框架，在连续动作空间下统一建模位置、能量、环境预报与任务约束，重点解决长时序决策中的状态表示、信用分配与策略稳定性问题。

  • 设计联合状态空间与多目标奖励函数，综合考虑驻留偏差、能量消耗与越界惩罚。
  • 提出时间-高度双重位置编码 THDPE，对多时刻、多高度层环境预报信息进行联合编码。
  • 引入任务条件化交叉注意力模块，使策略网络能够自适应聚焦与当前决策最相关的时空上下文。
  • 通过分段奖励、阶段性 shaping 与 GAE 设计缓解长时序 RL 中的 credit assignment 问题。

Reinforcement Learning for Domain-specific Code Agents

Recent Work

在 OpenROAD-Agent 项目中，我将强化学习方法用于领域代码智能体后训练，核心目标是利用真实工具执行结果提升模型的可执行性、API 准确性与自纠错能力，而不仅仅是文本级别的代码生成质量。

  • 系统比较 PPO 与 GRPO，在 32B 模型显存受限场景下选用更适合在线 RL 的 GRPO 方案。
  • 设计多维度 reward 函数，区分语法错误、API 幻觉调用与逻辑参数错误，并叠加自修复奖励。
  • 在 veRL 框架下实现 actor/rollout 解耦、vLLM 并行采样和基于 NCCL 的权重同步闭环。
  • 结合 KL 惩罚、reward 归一化、clip 控制与 LoRA 参数隔离，提升 RL 后训练稳定性。
