2025.11 - 2026.04

系统构建

  • 基于 Qwen3-Coder-32B 研发 OpenROAD-Agent，覆盖布局布线、时序分析等核心 EDA 流程。
  • 搭建 OpenROAD API 文档与代码模板知识库，引入 RAG 检索增强，缓解工具调用 hallucination。
  • 构建真实执行驱动的数据闭环，形成正确脚本、错误脚本与修复轨迹组成的垂直领域数据集。

训练与优化

  • 设计两阶段 SFT 流程，分别针对 TCL 代码生成与错误修复能力进行微调。
  • 对比 PPO 与 GRPO，最终采用更适合 32B 模型显存约束的 GRPO 方案。
  • 结合 KL 惩罚、reward 归一化、clip 机制与 LoRA 参数隔离，提升 RL 后训练稳定性。

结果表现

  • 脚本一次性执行成功率从 34% 提升至 76%。
  • 多轮自修复后整体任务完成率达到 94%。
  • 幻觉 API 调用率由 38% 降至 6%，平均修复轮次由 3.1 轮降至 1.3 轮。

RESEARCH

科研项目

Flagship Project

面向平流层飞艇长期驻留的时空前瞻强化学习路径规划

提出 STF-RL 路径规划框架，针对复杂时变风场与能量约束耦合场景，让智能体在长时序任务中具备环境前瞻性的变高度决策能力。

  • 构建“当前状态 + 未来风场”联合状态表示，显式引入多时刻、多高度层的预报风场。
  • 基于 Transformer 设计未来风场时空编码网络，并提出时间-高度双重位置编码 THDPE。
  • 设计任务条件化交叉注意力机制，与 PPO actor-critic 网络深度耦合优化。
  • 在多场景仿真中验证方法有效性，驻留性能与能量管理均优于 Baseline。
